\doxysubsection{Detailed Description}
The class \doxylink{classpaceval__c_graph}{paceval\+\_\+c\+Graph} is derived from \doxylink{classpaceval__c_registered_object}{paceval\+\_\+c\+Registered\+Object} and encapsulates the management of the node list as well as the identification and management of parallelization options for caching. 

For example the identify\+Multithread\+Opportunity function of the \doxylink{classpaceval__c_graph}{paceval\+\_\+c\+Graph} class looks for opportunities for parallelization. These opportunities are taken into account when processing the node list later with \doxylink{classpaceval__c_computation_a2cde08aa8ee7a3cd0da75fe4e296b6b9}{paceval\+\_\+c\+Computation\+::do\+Computation()}. Parentheses levels such as (1/x) in the function -\/sin(x\texorpdfstring{$\ast$}{*}cos(cos(x))\texorpdfstring{$^\wedge$}{\string^}(1/x) can then be calculated in parallel in advance in a separate thread, which saves significant time.
