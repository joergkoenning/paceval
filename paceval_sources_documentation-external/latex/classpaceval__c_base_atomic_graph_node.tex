\doxysubsection{Detailed Description}
The base class \doxylink{classpaceval__c_base_atomic_graph_node}{paceval\+\_\+c\+Base\+Atomic\+Graph\+Node} encapsulates the handling of the mathematical subfunctions, the so-\/called nodes in paceval. 

Data is stored for each node, which, among other things, indicates which operation is to be carried out on the node or which caching values are available. The deletion of the node itself and the deletion of all linked data of a node is secured by the \doxylink{classpaceval__c_registered_object}{paceval\+\_\+c\+Registered\+Object}.
