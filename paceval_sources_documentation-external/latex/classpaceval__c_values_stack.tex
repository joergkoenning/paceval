\doxysubsection{Detailed Description}
The class \doxylink{classpaceval__c_values_stack}{paceval\+\_\+c\+Values\+Stack} is derived from \doxylink{classpaceval__c_registered_object}{paceval\+\_\+c\+Registered\+Object} and encapsulates the management of values in a stack in all available floating point precisions long double, double and float. 

This includes managing the associated minimum and maximum value of the interval if the value is not trusted. This stack is required for the individual partial calculations in which the operations are carried out via the nodes. Since different threads use different stacks, each of which has its own values depending on its variables, we call this set of pointers to values a "{}values field"{}. And in addition, the values for variables are stored as pointers to arrays as specified by the user (e.\+g. call of \doxylink{paceval__main_8cpp_a4c4ea05d9207def52c8fa9ec3a68d51d}{paceval\+Library\+\_\+d\+Get\+Computation\+Result()}).
