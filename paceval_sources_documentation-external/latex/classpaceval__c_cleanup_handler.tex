\doxysubsection{Detailed Description}
The class \doxylink{classpaceval__c_cleanup_handler}{paceval\+\_\+c\+Cleanup\+Handler} is derived from \doxylink{classpaceval__c_registered_object}{paceval\+\_\+c\+Registered\+Object} and encapsulates the management of all objects created and registered in paceval. 

The class \doxylink{classpaceval__c_cleanup_handler}{paceval\+\_\+c\+Cleanup\+Handler} is therefore responsible for releasing all memory that was used for a \doxylink{classpaceval__c_computation}{paceval\+\_\+c\+Computation} object when \doxylink{paceval__main_8cpp_a3b5857bde82baf2316dd2a674436055f}{paceval\+Library\+\_\+\+Delete\+Computation()} is called.
