\doxysubsection{Detailed Description}
The class \doxylink{classpaceval__c_scanner}{paceval\+\_\+c\+Scanner} is derived from \doxylink{classpaceval__c_registered_object}{paceval\+\_\+c\+Registered\+Object} and encapsulates the exploration of mathematical strings as parts of the user\textquotesingle{}s function. 

The scanner itself is controlled by the \doxylink{classpaceval__c_syntactic_analysis}{paceval\+\_\+c\+Syntactic\+Analysis} object. The user\textquotesingle{}s function is run through once from left to right and all node objects are created in a single run, which is very performant.
