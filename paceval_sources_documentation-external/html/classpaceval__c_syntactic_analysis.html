Detailed Description

The class paceval_cSyntacticAnalysis is derived from paceval_cRegisteredObject and and encapsulates the analysis of the function with the paceval_cScanner object and the creation of node objects with the paceval_cGraph object.

The paceval_cSyntacticAnalysis and paceval_cScanner objects are only needed temporarily once to initiate the paceval_cComputation object and are then deleted again and their memory released:
