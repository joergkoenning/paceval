Detailed Description

The class paceval_cCleanupHandler is derived from paceval_cRegisteredObject and encapsulates the management of all objects created and registered in paceval.

The class paceval_cCleanupHandler is therefore responsible for releasing all memory that was used for a paceval_cComputation object when pacevalLibrary_DeleteComputation() is called.
