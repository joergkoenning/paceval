Detailed Description

The base class paceval_cBaseAtomicGraphNode encapsulates the handling of the mathematical subfunctions, the so-called nodes in paceval.

Data is stored for each node, which, among other things, indicates which operation is to be carried out on the node or which caching values are available. The deletion of the node itself and the deletion of all linked data of a node is secured by the paceval_cRegisteredObject.
