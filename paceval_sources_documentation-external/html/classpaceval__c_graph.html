Detailed Description

The class paceval_cGraph is derived from paceval_cRegisteredObject and encapsulates the management of the node list as well as the identification and management of parallelization options for caching.

For example the identifyMultithreadOpportunity() function of the paceval_cGraph class looks for opportunities for parallelization. These opportunities are taken into account when processing the node list later with paceval_cComputation::doComputation(). Parentheses levels such as (1/x) in the function -sin(x*cos(cos(x))^(1/x) can then be calculated in parallel in advance in a separate thread, which saves significant time.
